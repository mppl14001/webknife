Webknife - a web presentation framework (version x.x.x)

Make elegant web interfaces in minutes.

"Webknife is a web presentation framework for rapid user interface design on the client side, with an emphasis on HTML 5 that degrades gracefully in older browsers".

Webknife contains:

  • A gridless CSS framework
  • A reference syntax for production-ready HTML components
  • (In future versions) optional inclusion of extended JavaScript components (with jQuery bundled)

Webknife makes it easy for any web designer to make high quality interfaces more quickly, by selecting from predesigned 'components' for common design tasks.

Features

  • CSS reset
  • Forms
  • Buttons
  • Typography
  • Tables
  • Icons

How to get started

1. Download the framework 2. Edit the template pages 3. Customise and publish
  • Download a copy of the framework for each new web project
  • Keep the framework directory as part of your project
  • Edit template.html (a blank canvas for any new web pages)
  • Copy and paste from reference.html (production-tested HTML snippets)
  • Consult style.css (for alternative style options)
  • Attach other stylesheets and further customise pages
  • Delete unneeded files and directories (index, index.html, reference.html, framework/change-log.html)
  • Publish your web project

Predictable Browser Support

Every release supports all major browsers and operating systems, with 'graceful degradation' for old browsers.
